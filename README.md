# Item Borrowing Management System

## Description
This project is a simple web-based item borrowing management system developed as an individual academic project.
The system focuses on basic CRUD operations for managing item data and borrowing records.
It was created to understand fundamental web application development and database integration.

## Features
- Item data management (Create, Read, Update, Delete)
- Borrowing record management
- Basic web interface
- Database connection

## Technologies Used
- MySQL
- HTML
- CSS
- Visual Studio Code

## Project Type
Individual Academic Web Project
